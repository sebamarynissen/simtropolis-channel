# Simtropolis Channel Documentation

Welcome to the Simtropolis Channel for sc4pac.

## Coverage Report

View the [STEX Coverage Report](coverage-report/) to see which Simtropolis STEX files are available in sc4pac channels and which are missing.

The coverage report provides:
- Summary statistics on package coverage
- Breakdown by author and category
- Direct links to STEX profiles
- Detailed listings of missing packages

## About

This channel automatically polls Simtropolis STEX for new uploads and generates package files. Content creators can add a `metadata.yaml` file to their STEX uploads to provide package metadata.
